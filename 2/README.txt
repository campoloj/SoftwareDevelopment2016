The purpose of this project is to implement a given player interface and implement our own system

Assignment_2.py - the game interface
Assignment_2_tests.py - tests for the game interface
player.py - the given player interface
player_tests.py - tests for the given player interface

To use this program, pull the git repository
and enter the following command from the directory the git repo is located in:
python cs4500-lidauch-jcamp/2/take5/Assignment_2.py <number of players> <starting player>
where <number of players> and <starting player> are integers
and <starting player> is between 0-<number of players>

Begin reading for task 1 with player.py from the top to the bottom
Read player_tests.py from top to bottom next
Begin reading for task 2 with assignment_2.py from top to the bottom
Read assignment_2_tests.py from top to bottom
